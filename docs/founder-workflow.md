# Founder Workflow

## Daily

- Review `20-Inbox/`.
- Review blockers and open decisions.
- Pick one or two priorities.
- Create or update task packets.
- Update decisions when the plan changes.

## Weekly

- Review production status, roadmap, risks, metrics, and stale notes.
- Update the evidence registry for important claims.
- Convert useful inbox notes into decisions, PRDs, experiments, or references.

## Monthly

- Run a CEO strategy review.
- Run a CTO architecture review.
- Update fundraising story and market narrative.
- Review competitors, costs, security, and production readiness.
