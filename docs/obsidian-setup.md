# Obsidian Setup

Open the generated vault folder, usually `startup-brain-vault/`, in Obsidian.

Recommended usage:

- Keep Canvas files enabled.
- Use graph view for navigation, not as a source of truth.
- Do not commit local workspace files such as `.obsidian/workspace.json`.
- Use `20-Inbox/` for unclassified notes.
- Promote notes into area folders only after type, status, owner, confidence, and parent links are clear.
