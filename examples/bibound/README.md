# BiBound Example

This folder contains sanitized curated examples from the original BiBound Operating Brain.

It demonstrates how a real startup used the template concepts for evidence, CTO review, task packets, agent runs, and architecture modeling.

## Sanitization

- Secrets and tokens are not included.
- Private local paths were removed.
- Personal contact details were removed.
- Examples are illustrative and should not be treated as current production truth.

## Files

- `sample-evidence-registry.md`
- `sample-cto-review.md`
- `sample-task-packet.md`
- `sample-agent-run.md`
- `sample-architecture-model.md`

The `archive/` folder keeps a small sanitized source snapshot for historical reference only.
