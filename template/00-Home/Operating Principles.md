# Operating Principles

- One trusted place for operating context.
- Small decisions recorded early beat large reconstructions later.
- Evidence beats memory.
- Runtime truth beats documents for production status.
- Agents must reduce context load, not increase it.
- Templates should be useful before they are complete.
