# PRD - Operating Brain Quality Upgrade

Use this note when planning a quality pass over the operating brain.

## Quality Targets

- Clear source-of-truth map.
- No broken internal links.
- No secrets or private paths.
- Current production and security status.
- Context packs that reduce agent token usage.
- Evidence for critical claims.
