# PRD - Startup Operating Brain

## Problem

Founders and small teams lose context when strategy, code, agent output, PR evidence, production status, and security decisions live in separate places.

## Goal

Maintain one versioned operating brain that is readable in Obsidian, reviewable in GitHub, and safe for AI-agent workflows.

## Non Goals

- Replace source repositories.
- Replace runtime observability.
- Treat AI output as verified truth.

## Success Criteria

- Core notes have required frontmatter.
- Agents can work from context packs.
- Evidence is recorded for important claims.
- Local validators pass.
