# Production Readiness Checklist

- Runtime service deployed.
- Domain or endpoint configured.
- Secrets managed outside the repo.
- IAM least privilege reviewed.
- Database backups configured if applicable.
- Logs and alerts configured.
- Smoke test documented.
- Rollback path documented.
- Owner assigned.
